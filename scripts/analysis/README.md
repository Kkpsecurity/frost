# Analysis Scripts

Scripts for analyzing different aspects of the Laravel application to understand behavior, identify issues, and generate reports.

## Available Scripts

### `analyze_course_card_issues.php`
Analyzes issues with course card display and rendering.

**Usage:** `php scripts/analysis/analyze_course_card_issues.php`

### `analyze_course_issues.php`  
General course-related problem analysis and reporting.

**Usage:** `php scripts/analysis/analyze_course_issues.php`

### `analyze_instructor_dashboard.php`
Analyzes instructor dashboard functionality and performance.

**Usage:** `php scripts/analysis/analyze_instructor_dashboard.php`

### `analyze_instunit.php`
Analyzes instructor unit relationships and data integrity.

**Usage:** `php scripts/analysis/analyze_instunit.php`

## When to Use

- Investigating system issues or unexpected behavior
- Understanding data relationships and dependencies  
- Generating reports for stakeholders
- Performance analysis and optimization planning
- Before making significant system changes

## Output

Analysis scripts typically output:
- Summary reports
- Data statistics
- Issue identification
- Recommendations for fixes
- Performance metrics
