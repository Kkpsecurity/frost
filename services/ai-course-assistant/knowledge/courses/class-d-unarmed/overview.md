# Class D - Unarmed Security Officer Training

## Course Overview

The Class D Security Officer Training program prepares students for a career in unarmed security work. This comprehensive course meets state requirements for security licensure and provides practical skills needed in the field.

**Duration**: Typically 3-5 days (40 hours total)

## Course Objectives

By the end of this course, students will be able to:

- Understand the legal authority and limitations of security officers
- Conduct effective patrols and access control procedures
- Write clear, accurate incident reports
- Respond appropriately to emergencies
- Communicate professionally with the public and law enforcement
- Apply ethical standards and professional conduct in security work

## Core Topics Covered

### 1. Legal Authority and Liability

- Difference between security officers and law enforcement
- Citizen's arrest laws and limitations
- Trespassing and property rights
- Use of force concepts (verbal commands, hands-on control)
- Liability issues and avoiding legal problems

### 2. Patrol Procedures

- Types of patrols (foot, vehicle, stationary posts)
- Observation and reporting skills
- Access control and visitor management
- Key control and lock procedures
- Perimeter security checks

### 3. Report Writing

- Elements of an effective incident report
- Documenting observations objectively
- Chain of custody for evidence
- Using proper terminology
- Digital reporting systems

### 4. Emergency Response

- Fire safety and evacuation procedures
- Medical emergencies (when to call EMS, basic first aid if included)
- Natural disasters and severe weather
- Bomb threats and suspicious packages
- Active shooter response (Run, Hide, Fight concepts)

### 5. Communication Skills

- Radio communication and 10-codes
- De-escalation techniques
- Customer service in security
- Working with law enforcement and emergency services
- Testifying in court (if called as a witness)

### 6. Professionalism and Ethics

- Code of conduct for security officers
- Avoiding conflicts of interest
- Maintaining confidentiality
- Appearance and uniform standards
- Personal safety and situational awareness

## Certification Requirements

To successfully complete the Class D program:

- Attend all required training hours (100% attendance)
- Pass written examinations (typically 70% or higher)
- Demonstrate understanding of key concepts
- Complete any practical exercises or scenarios
- Maintain professional conduct throughout the course

## Career Applications

Class D certification qualifies students for positions such as:

- Retail security officers
- Corporate security guards
- Hospital security personnel
- Event security staff
- Access control specialists
- Patrol officers for residential/commercial properties

## State Licensing

Upon successful completion:

- Students receive a certificate of completion
- Certificate is used to apply for state security license
- Additional background check and application process required by state
- License must be renewed periodically (varies by state)

## Important Notes

- This course focuses on **unarmed** security work only
- Use of weapons (firearms, batons, chemical sprays) requires additional training
- Always follow your employer's specific policies and procedures
- Continue professional development after certification
